# Graph Library API

## Structures

### graph_node_t

Node of adjacency list.

### graph_t

Graph represented by adjacency lists.

## Functions

### graph_init

Initializes graph.

### graph_destroy

Frees graph memory.

### graph_add_vertex

Adds new vertex.

### graph_add_edge

Adds edge between two vertices.

### graph_remove_edge

Removes edge between two vertices.

### graph_are_adjacent

Checks whether vertices are adjacent.

### graph_dfs

Performs depth-first search.

### graph_bfs

Performs breadth-first search.
