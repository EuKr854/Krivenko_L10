Отчёт по лабораторной работе

Цель работы:

Разработка библиотеки на языке C для работы с графами на основе списков смежности.

Структура проекта:

- include — публичный интерфейс библиотеки
- src — реализация библиотеки
- tests — тесты на языке C
- python — тестирование через ctypes
- reports — отчёт и результаты проверок

Реализованные функции:

- инициализация графа
- освобождение памяти
- добавление вершины
- добавление ребра
- удаление ребра
- проверка смежности вершин
- обход графа DFS
- обход графа BFS

Сборка:

-make

Запуск тестов:

-test_graph.exe

Запуск Python-теста:

-py test_graph.py

Результаты

Библиотека успешно собирается и корректно выполняет операции над графом.
